## 🧩 About Me

데이터 흐름과 안정성을 기준으로  
로직과 책임을 분리하는 백엔드 개발을 지향합니다.

금융 시스템 운영 경험을 통해 대규모 배치 처리와 데이터 정합성 관리 방식을 이해했고,  
웹 서비스 개발 과정에서는 팀 협업 기반의 역할 분리, 예외 처리, MSA 구조 설계를 경험했습니다.

안정적인 데이터 처리와 확장 가능한 구조를 중심으로  
실제 개발 환경에서 신뢰할 수 있는 서비스를 만들어 가겠습니다.
